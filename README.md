# Weather MCP Israel

שרת MCP לשליפת תחזית מזג האוויר בישראל באמצעות אוטומציה של דפדפן עם Playwright.

---

## תיאור הפרויקט

בפרויקט זה פותח שרת MCP המאפשר למודל שפה לקבל מידע עדכני על מזג האוויר בערים בישראל.

במקום להשתמש ב־API ייעודי, הפרויקט מבצע אוטומציה של דפדפן:

1. פותח את אתר Weather2Day.
2. מזין את שם העיר המבוקשת.
3. בוחר את העיר מתוך רשימת ההשלמה האוטומטית.
4. מחלץ את נתוני התחזית מהדף.
5. מחזיר את המידע למודל לצורך ניסוח תשובה מלאה.

---

## מטרות לימודיות

הפרויקט מדגים:

- פיתוח שרת MCP עצמאי.
- חשיפת כלים למודל שפה.
- שליטה בדפדפן באמצעות Playwright.
- חילוץ מידע מאתר אינטרנט.
- שימוש במידע שנשלף לצורך העשרת הקונטקסט של המודל.

## הכלים שמומשו בפרויקט

### open_weather_forecast_israel

פותח את אתר התחזית של Weather2Day.

### enter_weather_forecast_city_israel

מקבל שם עיר ומזין אותו בשדה החיפוש.

### select_weather_forecast_city_israel

בוחר את העיר הראשונה מתוך רשימת ההשלמה האוטומטית.

### get_weather_data_israel

מחלץ את נתוני מזג האוויר מהדף ומחזיר אותם למודל.

## הוראות התקנה

### התקנת התלויות

```bash
uv sync
```

### התקנת הדפדפן של Playwright

```bash
uv run playwright install chromium
```

### יצירת קובץ env

יש ליצור קובץ בשם `.env` ולהוסיף בו:

```env
OPENAI_API_KEY=your_openai_api_key
```
